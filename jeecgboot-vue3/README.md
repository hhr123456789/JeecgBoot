# 告警处理模块更新说明

## 新增文件
1. `src/views/EnergyAlarm/Alarm_Handling/index.vue`
   - 创建告警处理主视图文件
   - 实现左侧部门树选择功能
   - 实现告警处理列表和功能

## 功能说明

### 1. 部门树选择
- 左侧实现部门树形选择控件
- 支持部门搜索功能
- 支持展开/收起功能

### 2. 筛选功能
- 时间范围选择（今日/本周/本月/自定义）
- 告警类型筛选（全部/超负荷/异常运行/故障）
- 告警状态筛选（全部/未处理/处理中/已处理）

### 3. 统计展示
- 总告警数统计
- 未处理告警统计
- 处理中告警统计
- 已处理告警统计

### 4. 告警列表
- 支持分页
- 支持批量选择
- 告警等级标识（高/中/低）
- 告警状态标识（未处理/处理中/已处理）
- 支持单条告警处理
- 支持查看详情
- 支持查看历史

### 5. 告警处理
- 弹窗形式
- 支持选择处理方式（重置/维修/忽略）
- 支持填写处理说明

## 样式说明
- 采用 Tailwind CSS 进行样式管理
- 响应式布局设计
- 统一的颜色系统
  - 未处理：绿色
  - 处理中：橙色
  - 已处理：绿色

## 注意事项
1. 目前使用静态数据，后续需要对接后端API
2. 部分功能（如详情查看、历史记录）需要进一步实现
3. 批量处理功能需要后续完善

# 仪表板分析页面更新说明

## 新增文件
1. `src/views/dashboard/analysis_new/index.vue`
   - 创建仪表板分析主视图文件
   - 实现页面整体布局
   - 集成各个子组件

2. `src/views/dashboard/analysis_new/components/TotalStatistics.vue`
   - 创建总体统计卡片组件
   - 显示总能耗、总用水量、总用气量、综合能耗等关键指标
   - 支持同比数据展示

3. `src/views/dashboard/analysis_new/components/EnergyTrend.vue`
   - 创建能源趋势图组件
   - 使用echarts实现折线图
   - 支持多种能源类型的趋势对比

4. `src/views/dashboard/analysis_new/components/EnergyDistribution.vue`
   - 创建能源分布图组件
   - 使用echarts实现饼图
   - 展示不同能源类型的占比分布

5. `src/views/dashboard/analysis_new/components/RealTimeMonitor.vue`
   - 创建实时监控组件
   - 显示设备运行状态列表
   - 支持设备状态、功率、能耗等数据展示

## 功能说明

### 1. 总体统计
- 总能耗统计（kWh）
- 总用水量统计（m³）
- 总用气量统计（m³）
- 综合能耗统计（tce）
- 支持同比数据展示和趋势指示

### 2. 能源趋势
- 支持日/月/年时间维度切换
- 多种能源类型趋势对比
- 平滑曲线展示
- 支持图例切换和提示框

### 3. 能源分布
- 饼图展示各类能源占比
- 支持能源类型筛选
- 图例显示具体百分比
- 悬浮提示详细数据

### 4. 实时监控
- 设备运行状态实时展示
- 支持状态标签颜色区分
- 显示实时功率和累计能耗
- 记录设备运行时长

## 样式说明
- 采用 Tailwind CSS 进行样式管理
- 响应式布局设计
- 统一的颜色系统
  - 运行中：绿色
  - 已停止：灰色
  - 故障：红色
- 图表样式统一，确保视觉一致性

## 注意事项
1. 目前使用静态数据，后续需要对接后端API
2. 图表组件支持窗口大小自适应
3. 所有数值展示保留合适的小数位数
4. 表格数据支持自定义列宽和对齐方式
